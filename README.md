# Sales_Analysis (Excel Project)

## Project Overview
This project is made in Excel to analyze sales data in an easy way.
It helps to understand how sales are growing and which products or regions are performing better.

## Objectives
* Find total sales and profit
* Understand sales trends
* Identify top products and regions
* Create a simple dashboard

## File Included
* Sales_Analysis.xlsx (Excel file with data and dashboard)

## Features
* Simple and clean data
* Use of pivot tables
* Charts for better understanding
* Dashboard with filters

## Tools Used
* Microsoft Excel
  * Pivot Tables
  * Charts
  * Slicers

## Dashboard
The dashboard shows:
* Total Sales
* Total Profit
* Sales by Product
* Sales by Region
* Monthly Sales
You can use filters to see different data.

## How to Use
1. Open the Excel file
2. Go to the Dashboard sheet
3. Use filters to explore data
4. Check charts for insights
